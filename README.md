# homebrew-archi2likec4

Homebrew tap for [archi2likec4](https://github.com/Lenivvenil/archi2likec4) — a tool that converts coArchi XML (ArchiMate) repositories to LikeC4 `.c4` files.

## Installation

```bash
brew tap Lenivvenil/archi2likec4
brew install archi2likec4
```

## Usage

```bash
archi2likec4 --help
archi2likec4 --version
```

## Upgrade

```bash
brew upgrade archi2likec4
```

## Uninstall

```bash
brew uninstall archi2likec4
brew untap Lenivvenil/archi2likec4
```
